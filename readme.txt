== Description ==
Plugin shows Twitter users who recently tweeted links from your blog. Plugin also shows Title of blog posts that were tweeted.

== Installation ==
1. Upload retweeters.zip to '/wp-content/plugins/' directory and unzip it.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. If you are using Wordpress Widgets, go the "Widgets" menu, and from "Available Widgets" menu, drag and drop the "Retweeters" widget onto the desired "Sidebar"
